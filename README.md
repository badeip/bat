### Bat
Combined `cat`, `dd` and `hexdump`


###Requirements:
  - `boost_program_options` (apt-get install libboost-all-dev or some subset thereof)
  - Linux/MacOS: `make` `g++` / `clang++`
  - Windows: use the included Visual Studio project file.
